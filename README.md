 - API available on port **80**
 - Adminer available on port **8080**

### Requirement
 - docker 
 - make

> BUILD dev mode (run alse adminer) with
```sh
$ make devbuild
```
> RUN in dev mode (run also adminer) with
```sh
$ make devup
```
> Make database with
```sh
$ make db
```
> Insert test data with
```sh
$ make data
```
> DOWN in devmode with
```sh
$ make devdown
```
